# Insight Master Demo

Threads分析AIツール「Insight Master」のフルデモ版です。  
セットアップ不要 - ブラウザだけで全機能を体験できます。

## 🚀 デモURL

**https://goroyattemiyo.github.io/insight-master-demo/**

## ✨ 機能

- 📊 ダッシュボード（フォロワー推移・成長スコア・デモグラフィック）
- 📈 アナリティクス（投稿分析・バズパターン・時間帯ヒートマップ）
- ✍️ AI投稿作成（通常・スレッド・分析ベース）
- 🔍 キーワード検索
- 📝 下書き管理
- 🤖 Gemini API連携（オプション）

## 🔑 Gemini API キー（オプション）

設定画面からGoogle Gemini APIキーを入力すると、デモデータの代わりにリアルタイムAI分析が利用できます。

1. [Google AI Studio](https://aistudio.google.com/apikey) でAPIキーを取得
2. アプリの設定画面でキーを入力
3. AIによるリアルタイム分析・投稿生成が有効化

**キーがなくても全画面をサンプルデータで体験できます。**

## 🏗️ 技術

- 単一HTMLファイル SPA（ビルドツール不要）
- Vanilla JavaScript
- LocalStorage によるデータ永続化
- GitHub Pages デプロイ
